# Gasto Ituzaingó (APK via GitHub Actions)

1. Crea un repo nuevo en GitHub y subí **todo este contenido**.
2. En GitHub ve a **Actions** y habilitá workflows si te lo pide.
3. Hacé un push a la rama `main`.
4. Al terminar el workflow, descargá el APK desde **Actions → Gasto-Ituzaingo-APK** (Artifacts).

También podés abrir el proyecto en Android Studio localmente y compilar.
